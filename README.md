# BPL Interpreter (Basic Programming Language)

BPL — это строго типизированный язык программирования и интерпретатор, разработанный в рамках индивидуального проекта «Путь кода до исполнения: интерпретация». Проект создан как наглядный образовательный инструмент, демонстрирующий внутреннее устройство интерпретаторов.

## 📋 О проекте
Цель проекта — приоткрыть «черный ящик» процесса исполнения кода. BPL позволяет проследить трансформацию обычного текста в работающую логику программы через классические этапы обработки данных.

### Основные особенности:
- **Строгая типизация:** Использование системы `TypedObject` для безопасной работы с данными.
- **Управление памятью:** Реализован механизм `Environment` для корректной работы с областями видимости, хранения переменных и функций.
- **Паттерн Visitor:** Гибкая архитектура обхода абстрактного синтаксического дерева (AST).
- **Поддержка функций:** Возможность использования встроенных функций и написания алгоритмических задач.

## 🏗 Архитектура
Интерпретатор BPL работает как трехэтапный конвейер:
1. **Лексический анализ (Lexer):** Разбивает исходный текст на минимальные смысловые единицы — токены.
2. **Синтаксический анализ (Parser):** На основе грамматики языка строит древовидную структуру (AST), отражающую иерархию команд.
3. **Интерпретация (Interpreter):** Выполняет программу, поочередно обрабатывая узлы дерева с помощью паттерна «Visitor».

## 🚀 Запуск и использование
Для работы интерпретатора необходима **Java 21** или выше.

### Инструкция:
1. Перейдите в раздел [Releases](https://github.com/Field-Boy/BplInterpreter/releases) и скачайте актуальную версию JAR-файла (например, `BplInterpreter-1.0.1`).
2. Подготовьте файл с кодом на языке BPL.
3. Запустите интерпретатор через терминал:
   ```bash
   java -jar <путь_к_интерпретатору> <путь_к_файлу> [флаги]

Доступные флаги отладки:

--log-lexer — отображает список токенов, полученных после лексического анализа.

--log-parser — визуализирует структуру AST-дерева вашей программы.

### 📂 Тестовые примеры
В репозитории есть папка test-scripts, в которой собраны примеры кода. Они демонстрируют синтаксис языка: объявление переменных, циклы, условия и работу с функциями. Это отличная точка входа для изучения возможностей BPL.
